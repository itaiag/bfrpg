# BFRPG Project — Claude Memory

## Project Overview
Quarto book site for Basic Fantasy RPG (BFRPG) rules. English source at root, Hebrew translation in `he/`. Built output goes to `docs/` (GitHub Pages).

## Build Commands
```
quarto render          # English site → docs/
node build-he.js       # generate he/_quarto.yml + quarto render he/
node build-he.js --preview  # live preview of Hebrew site (hot-reload)
```
**Important**: `--profile he` merges chapters additively with the root `_quarto.yml`. Always render `he/` as a standalone project instead.

## Architecture Decisions

### Hebrew Quarto Build
- `he/` is its own standalone Quarto project
- `he/_quarto.yml` is auto-generated by `build-he.js` (gitignored)
- Uses `lang: he-IL` (not `lang: he` — conflicts with `he/` directory name)
- `output-dir: ../docs/he`

### Dice Notation
- Keep English `d` notation (1d6, 2d8, etc.) — do NOT use Hebrew ק
- Reason: ק is RTL; bidi algorithm scrambles surrounding digits in Hebrew paragraphs
- Hebrew RPG publications standardly use English `d` notation

### Unit Conversion
- Imperial units were **replaced entirely** with metric (no parenthetical append)
- Game-friendly rounding: feet×0.3 (→ nearest 0.5m), miles×1.5 (→ nearest 0.5km), pounds×0.45 (→ nearest 1kg)

### GM Gender
- Game Master (מנחה) is always feminine in Hebrew

## Key Files
| File | Purpose |
|------|---------|
| `build-he.js` | Generates `he/_quarto.yml` and runs `quarto render he/` |
| `translator/lang-switch.js` | Client-side EN↔HE toggle button |
| `translator/glossary.json` | RPG term translations |
| `translator/glossary-spells.json` | Spell name translations |
| `translator/glossary-monsters.json` | Monster name translations |
| `custom-rtl.css` | RTL layout, Frank Ruhl Libre font |
| `he/` | Translated QMD files (git-tracked) |
| `docs/he/` | Built Hebrew HTML site (git-tracked for Pages) |

- `he/.quarto/` — Quarto cache, gitignored
- `he/` QMD files — tracked
- `docs/he/` — tracked (GitHub Pages)
